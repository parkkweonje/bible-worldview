Open & Public-Domain Scriptures

자료실 · 공개 성경 다운로드

저작권이 만료되었거나 자유 이용이 허락된 공개(퍼블릭 도메인) 성경만 모았습니다. 아래 카드를 누르면 각 번역본을 합법적으로 제공하는 공식 출처로 이동해 TXT·PDF·EPUB 등으로 내려받을 수 있습니다.

이용 안내. 여기 소개하는 성경은 모두 저작권이 소멸했거나(대개 발행·저자 사망 기준 보호기간 경과) 배포가 자유롭게 허락된 판본입니다. 파일은 이 사이트가 아니라 각 공개 배포 기관 (Project Gutenberg, eBible.org 등)에서 직접 받습니다. 다만 대부분의 한국어 번역본(개역개정·새번역 등)은 대한성서공회의 저작권이 있어 전문을 무단 배포할 수 없으므로, 한국어는 자유 이용이 확인된 출처만 안내합니다. 이용 지역의 저작권법을 한 번 더 확인하시길 권합니다.

영어 공개 성경

저작권이 소멸한 대표 영어 번역본. 카드를 누르면 공식 배포처의 다운로드 페이지로 이동합니다.

King James Version (KJV) 퍼블릭 도메인

1611년 초판 · 영어 흠정역. 서양 문학·신앙에 가장 큰 영향을 준 고전 번역. TXT·EPUB·Kindle·HTML로 제공.

Project Gutenberg · ebook #10

World English Bible (WEB) 퍼블릭 도메인

ASV(1901)를 현대 영어로 개정한 판본. 저작권을 아예 포기(PD)해 자유롭게 복사·배포 가능. USFM·PDF·EPUB·앱 등 제공.

eBible.org

American Standard Version (ASV) 퍼블릭 도메인

1901년 미국 표준역. 직역에 충실해 연구용으로 널리 쓰이는 고전 판본. 여러 형식으로 다운로드 제공.

eBible.org

Young's Literal Translation (YLT) 퍼블릭 도메인

1898년 로버트 영의 초직역본. 원문 시제·어순을 최대한 살린 판본으로 원어 대조에 유용.

eBible.org

Douay-Rheims Bible 퍼블릭 도메인

라틴 불가타에서 번역한 영어 가톨릭 고전 성경. TXT·EPUB 등으로 제공.

Project Gutenberg

Darby Translation (1890) 퍼블릭 도메인

존 넬슨 다비의 직역 성경. 원문에 밀착한 번역으로 대조 연구에 도움이 됩니다.

eBible.org

한국어 · 원어(히브리어/헬라어)

한국어 최신 번역본은 대부분 저작권 보호 대상입니다. 자유 이용이 확인된 출처만 안내합니다.

한국어 성경 (자유 이용본 안내) 자유 이용

eBible.org의 한국어 목록에서 배포가 허락된 판본을 확인·다운로드할 수 있습니다. 저작권 있는 번역본은 여기서 배포하지 않습니다.

eBible.org · Korean

Open Scriptures Hebrew Bible CC 라이선스

모음·강세까지 표기된 히브리어 구약(WLC 기반) 원문 데이터. 형태소 분석 포함, 자유 이용 라이선스.

OpenScriptures (GitHub)

Greek New Testament (Byzantine) 퍼블릭 도메인

비잔틴 다수사본 헬라어 신약 원문. 파싱 데이터와 함께 자유롭게 내려받아 연구에 활용할 수 있습니다.

Byzantine Majority Text

여러 언어·형식 통합 다운로드

수백~수천 개 언어의 공개 성경을 한곳에서 형식별로 받을 수 있는 공식 아카이브입니다.

eBible.org 무료 배포

1,000개 이상 언어의 공개·자유 이용 성경을 USFM·PDF·EPUB·Sword 모듈 등으로 제공하는 대표 아카이브.

ebible.org

getBible API 무료 · JSON

다양한 공개 번역본을 JSON으로 내려받거나 프로그램에 바로 연동할 수 있는 무료 성경 데이터 API.

getbible.net

Project Gutenberg 퍼블릭 도메인

저작권이 만료된 성경·경전을 TXT·EPUB·Kindle 등으로 제공하는 전자책 도서관.

gutenberg.org

Wikisource — Bible 자유 이용

여러 공개 번역본의 본문을 온라인으로 읽고 복사할 수 있는 자유 문헌 도서관.

wikisource.org
📥 다운로드 방법. 각 카드를 누르면 해당 기관의 공식 페이지가 새 창으로 열립니다. 그 페이지에서 원하는 형식(TXT·PDF·EPUB·앱 등)을 선택해 받으시면 됩니다. 링크가 열리지 않는 경우 기관 사정으로 주소가 바뀌었을 수 있으니 연락으로 알려 주시면 갱신하겠습니다.
